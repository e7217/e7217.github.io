이 저장소를 개인 기술 블로그와 프로젝트 기록 공간으로 구성하는 작업입니다.

## 목표

- Markdown 기반으로 개발 기록을 빠르게 작성한다.
- 스마트팩토리와 AI를 중심 주제로 유지한다.
- 프로젝트 기록이 자연스럽게 포트폴리오 자료로 이어지게 한다.
- GitHub Pages 배포 흐름을 유지한다.

## 현재 범위

- 홈, 블로그, 프로젝트, 카테고리, 태그, About 페이지 구성
- 큰 카테고리 중심의 초기 분류 전략 적용
- 첫 개발 기록 포스트 추가
